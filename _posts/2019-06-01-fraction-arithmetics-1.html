Evaluate without a calculator.

  1. $$\displaystyle\frac{3}{12} + \displaystyle\frac{5}{12}$$
  2. $$\displaystyle\frac{11}{6} + \displaystyle\frac{2}{6}$$
  3. $$\displaystyle\frac{5}{2} + \displaystyle\frac{7}{3}$$
  4. $$\displaystyle\frac{2}{7} + \displaystyle\frac{5}{17}$$
{% capture solution %}
  1. $$\fraction{3}{12} + \fraction{5}{12} = \fraction{8}{12} = \fraction{2}{3}$$
  2. $$\fraction{19}{12} - \fraction{4}{3} = \fraction{19}{12} - \fraction{16}{12} = \fraction{3}{12} = \fraction{1}{4}$$
  3. $$\fraction{5}{8} \times \fraction{3}{2} = \fraction{15}{16}$$
  4. $$\fraction{6}{5} \div \fraction{2}{8} = \fraction{6}{5} \times \fraction{8}{2} = \fraction{48}{10} = \fraction{24}{5}$$
  5. $$\fraction{14}{4} - \fraction{8}{6} = \fraction{42}{12} - \fraction{16}{12} = \fraction{26}{12} = \fraction{13}{6}$$
  6. $$\fraction{1}{4} \div \left( \fraction{5}{8} + \fraction{2}{4} \right) = \fraction{1}{4} \div \left( \fraction{5}{8}+\fraction{4}{8} \right) = \fraction{1}{4} \div \fraction{9}{8} = \fraction{1}{4} \times \fraction{8}{9} = \fraction{8}{36} = \fraction{2}{9}$$
