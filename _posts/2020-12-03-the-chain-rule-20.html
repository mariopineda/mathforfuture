Given $f(x)=(2x+1)^5 (x^3-x+1)^4$;

  1. Sketch $f(x)$. Use Desmos to help you visualize the graph before sketching.
  2. Predict $f'(x)$ by superimposing a sketch of it onto the graph of $f(x)$.
  3. Differentiate $f(x)$. Answer in fully factoried form. {% include_relative hint-modal1.html content="Use the Product Rule before using the Chain Rule." %}.
  4. Solve $f'(x)=0$ using the Zero Product Theorem by graphically solving each factor for $x$ using Desmos.
