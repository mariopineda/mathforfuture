This website is a curated repository of high school math problems aligned with the Alberta Education program of Study for grade 10-12 mathematics. The primary purpose of this web site is for me to use it with my own students, but it is also an experiment in 'open source' math education which means that it is publicly avaliable for anyone else to use as well.

FAQ

Who made this web site?

I am an Edmonton-based high school math teacher.

I found an mistake in a question or solution

Thank you for finding it. You can report the error using the "Report Errors" link on the bottom of the page where the practice problem is.

Why is a question missing a solution?

This site is permanently under construction and the development of the practice problems is an ongoing "one man show". Sometimes I fall behind a bit. Rest assure that I am always working on new practice problems and solutions as fast as I can. Check back again and new problems and solution will have been posted.
